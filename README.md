# Neuron - Second Brain

A modern knowledge management system built with React, TypeScript, and Vite.

## Features

- 📝 Block-based editor (Notion-style)
- 🎨 Beautiful UI with light/dark themes
- 📁 Folder organization
- 🔍 Search and filter
- 📌 Pin important cards
- 🤖 AI integration
- 🌐 Multi-language support (EN/ZH)

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand
- TipTap Editor

## Development

\`\`\`bash
npm install
npm run dev
\`\`\`

## Build

\`\`\`bash
npm run build
npm run preview
\`\`\`
